# Ecommerce Backend

Minimal Node.js, Express, and TypeScript API.

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

The server starts without MongoDB, JWT, or Stripe configuration.

## Scripts

```bash
npm run dev      # start development server
npm run build    # compile TypeScript
npm start        # run compiled server
```

## API Base

`http://localhost:5000/api`

## Routes

- `GET /`
- `GET /api/health`
